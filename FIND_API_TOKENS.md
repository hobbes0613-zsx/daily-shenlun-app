# Coze API Tokens 位置详细查找指南

## 🔍 在哪里找到 API Tokens

Coze 的界面可能会更新，API Tokens 可能在以下位置：

---

## 📍 位置 1：个人中心（最可能）

### 路径
```
Coze 首页 → 右上角头像 → 个人中心 → API 管理 / API Tokens
```

### 详细步骤
1. 登录 https://www.coze.com
2. 看页面右上角，找到你的头像
3. 点击头像，出现下拉菜单
4. 寻找以下选项之一：
   - ✅ **个人中心** / **Personal Center**
   - ✅ **设置** / **Settings**
   - ✅ **API 管理** / **API Management**
   - ✅ **API 密钥** / **API Keys**

---

## 📍 位置 2：开发者菜单

### 路径
```
Coze 首页 → 顶部菜单 → 开发者 / Developer → API Tokens
```

### 详细步骤
1. 登录 Coze
2. 看页面顶部导航栏
3. 寻找菜单项：
   - ✅ **开发者** / **Developer**
   - ✅ **API 文档** / **API Docs**
   - ✅ **API 管理** / **API Management**

---

## 📍 位置 3：侧边栏

### 路径
```
Coze 首页 → 左侧菜单 → Settings → API Tokens
```

### 详细步骤
1. 登录 Coze
2. 看页面左侧边栏
3. 寻找：
   - ✅ ⚙️ **Settings** / **设置**
   - 点击后找 **API Tokens**

---

## 📍 位置 4：用户资料页

### 路径
```
Coze 首页 → 点击你的名字 → 个人资料 → API Tokens
```

---

## 📍 位置 5：帮助中心 / 文档

### 路径
```
Coze 首页 → 帮助中心 / 文档 → API 文档 → API Keys
```

---

## 🔍 查找技巧

### 技巧 1：搜索功能
1. 按 `Ctrl+F` (Windows) 或 `Cmd+F` (Mac)
2. 搜索关键词：
   - `API Token`
   - `API Key`
   - `密钥`
   - `Token`

### 技巧 2：查看所有菜单
- 把鼠标悬停在所有菜单项上
- 看是否有下拉子菜单
- 特别注意包含 "API"、"设置"、"个人" 的菜单

### 技巧 3：检查用户权限
- 确保你的账号有 API 访问权限
- 可能需要完成实名认证
- 可能需要绑定手机号

## 🆘 如果还是找不到

### 方案 1：搜索 Coze 文档
1. 访问：https://www.coze.com/docs
2. 搜索 "API Token" 或 "API Key"
3. 文档中会提供准确位置

### 方案 2：联系 Coze 客服
- 在 Coze 网站找客服入口
- 或发送邮件咨询

### 方案 3：使用浏览器插件
1. 按 `F12` 打开开发者工具
2. 查看网络请求
3. 查找包含 "token" 或 "key" 的请求
